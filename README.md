# Blinkit Sales Analysis using Python

## Overview

This project analyzes Blinkit grocery sales data using Python to identify sales trends, outlet performance, and customer purchasing patterns through data cleaning, KPI analysis, and visualization.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- KPI Calculation
- Sales Trend Analysis
- Business Visualizations

## Files

- `Blinkit Analysis in Python.ipynb` – Jupyter notebook containing the complete analysis.
- `blinkit_data.csv.xlsx` – Dataset used for the project.
